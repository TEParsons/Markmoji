# `markmoji.handlers.IPAHandler`

Handler to format text written in International Phonetic Alphabet, adding a link to ipa-reader.xyz


#### Parameters
label (str)
:    Text to display.

link (str)
:    IPA text to pronounce in ipa-reader.xyz. Leave blank to use text from label.

## Example
### Markdown:
```
🗣️<aɪpʰieɪ>
```
### Result:
🗣️<aɪpʰieɪ>
