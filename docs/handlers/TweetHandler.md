# `markmoji.handlers.TweetHandler`

Handler for an embedded tweet.

### Parameters
label (str)
:    Unused as embedded tweets don't have alt text

link (str)
:    Link to the tweet to embed

## Example
### Markdown:
```
🐦<https://twitter.com/edballs/status/63623585020915713>
```
### Result:
🐦<https://twitter.com/edballs/status/63623585020915713>
